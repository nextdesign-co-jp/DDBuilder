==============
Jクラスレポート
==============

本製品は同梱の「ソフトウエアの使用条件」に同意された方のみご使用いただけます。

１．インストール方法

(1)ステップ１
ダウンロードしたファイルを解凍してください。

(2)ステップ２
このステップはC:\以外の場所に解凍した場合にだけ必要です。
次のファイルをテキストエディタで編集して、解凍場所を指定してください。
jp-co-nextdesign-jclassreport\USER_DIR.bat

２．起動方法
jp-co-nextdesign-jclassreport\report.bat を起動（ダブルクリック）してください。

３．アンインストール方法
jp-co-nextdesign-classreportフォルダを削除してください。

４．確認済みの稼動環境
Java SE6 + Windows7 Professional

５．対象となるJavaソース
すべて確認はできませんが
Java 1.3
Java 1.4
Java 1.5
Java 1.6
は処理できます。

６．作成ドキュメントの確認済み環境
Microsoft Excel2010

７．操作手順
メニュー画面が表示されたら
（１）設定ボタンを押下して必要項目を設定します。（必要な場合のみ）
（２）ソースディレクトリを指定します。（例）xxx\projectname\src
（３）レポート出力先を指定します。(例）yyy\report
（４）レポート作成ボタンを押下します。(例）yyy\report以下にExcelファイルが作成されます）
